# Harshit Dubey — Developer Portfolio

A high-performance, interactive 3D developer portfolio designed with a "Graphite Lab" aesthetic. This project showcases my skills in full-stack development, machine learning, and data analytics.

## 🚀 Features
- **3D Hero Section**: Interactive particle network and rotating icosahedron built with `Three.js` that reacts to scroll and mouse movement.
- **Premium UI/UX**: Custom interactive cursor glow, cyberpunk glitch text effects, and glassmorphic elements.
- **3D Tilt Project Cards**: Vanilla JavaScript physics calculation that adds a tangible depth response to project cards when hovered.
- **WhatsApp Integration**: Contact form that dynamically generates a WhatsApp message payload and opens the desktop or web client.
- **Zero Dependencies (mostly)**: Built primarily with standard HTML, CSS, and Vanilla JS for maximum performance.

## 💻 Tech Stack
- **HTML5 & CSS3** (Custom "Graphite Lab" design system)
- **Vanilla JavaScript** (DOM manipulation, 3D tilt math, scroll reveal observers)
- **Three.js** (WebGL 3D Canvas rendering)
- **Vite** (Development server and fast build tooling)

## 🛠️ Local Development

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Harshitdubey-lab/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
